## About AI × Web3 School

AI × Web3 School 是一个面向 builders 的开源共学计划，由 [LXDAO](https://lxdao.io/) 和 ETHPanda 发起。聚焦 AI 与 Web3 的交叉领域：Agent 工作流、链上工具调用、Agent 钱包与权限、加密支付、可验证 AI、AI 安全等前沿话题。

- 📖 **Handbook（知识地图）**：https://aiweb3.school/zh/handbook/
- 🎓 **WCB 课程页**：https://web3career.build/zh/programs/AI-Web3-School
- 💬 **Telegram 社区**：https://t.me/aiweb3school
- 🐦 **X（Twitter）**：https://x.com/aiweb3school

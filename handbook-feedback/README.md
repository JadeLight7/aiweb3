# Handbook Feedback

> 学习过程中发现的 Handbook 问题：typo、概念不清、材料过时、结构建议等。
>
> 每条反馈独立一个文件，命名格式：`YYYY-MM-DD-<topic>.md`

## Feedback Template

```markdown
# Feedback: [Title]

## Source
- Handbook page: [URL]
- Found during: [daily/YYYY-MM-DD.md]

## Problem
<!-- 描述你遇到的问题 -->

## Suggested Fix
<!-- 建议如何修改 -->

## Additional Context
<!-- 截图、代码示例、参考链接等 -->
```

## Why Public Feedback?

- 本仓库为公开仓库，Handbook 维护者可以直接索引和拉取反馈
- 每条反馈可被社区讨论和改进
- 开源共学的核心：问题驱动 → 沉淀 → 回馈
